{%- comment -%} @imqueue glyph. Emits an equal width/height (default 16) purely to give the SVG a 1:1 intrinsic ratio, so CSS `height:100%; width:auto` renders it square. The value is kept small (<= brand name line height) so it never forces the lockup line taller than the text block — the glyph then stretches to name-only or name+slogan height via CSS. Pass `size` to force explicit dimensions for standalone use. {%- endcomment -%} {%- assign _sz = size | default: 16 -%}
