IMQ is an Intercommunication Messaging Queue Framework for service oriented back-end architectures.

There are several key capabilities:

  • Massaging Queue as a base communication protocol between services
  • RPC-like programming interface
  • Rapid application development command line interface

These provides an ability for developers to create back-end services (or micro-services as a special case of SOA) very fast and focus only on functionality implementation without need to take care of low-level implementation.

IMQ protocol takes care about providing all necessary capabilities for building distributed service-oriented application which are described below.
